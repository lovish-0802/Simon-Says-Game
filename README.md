# Simon Says Game 🧠🎮

An interactive memory-based game inspired by the classic Simon Says, built using JavaScript.

## 🚀 Features
- Random color sequence generation
- Increasing difficulty with each level
- User input tracking and validation
- Game over detection and restart option
- Visual feedback for user actions

## 🧠 How It Works
- The game generates a random sequence of colors
- The player must repeat the sequence in the correct order
- Each successful round increases the sequence length
- The game ends if the user makes a mistake

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

## ▶️ How to Play
1. Press any key to start the game
2. Watch the sequence of colors
3. Repeat the sequence by clicking the buttons
4. Continue as long as you can without making mistakes
